SNAPSHOT RADIO CHANNELS

Read all 8 channel slots from the connected radio and save them to the vault database. Includes channel names, PSKs, and settings. Use before hotswapping to preserve the current state.

HOW BACKUP WORKS
① SNAPSHOT — reads current radio channels into vault DB.
② IMPORT TO VAULT — copies channel PSKs from backup into the server-side decryption vault.
③ HOTSWAP — writes any vaulted channel to a specific radio slot, then optionally reboots.
④ RESTORE — hotswap each backup channel back to its original slot to undo changes.
