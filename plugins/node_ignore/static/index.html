HOW IT WORKS
Ignored nodes are hidden from the node grid, map, DM panel, and all UI views.
The node still operates on the mesh. Its data is preserved — ignore is reversible.
The ignore flag is stored server-side in SQLite and survives dashboard restarts.
The bridge stamps _ni_ignored into live node data and filters it before rendering.
