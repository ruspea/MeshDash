REBOOTDevice disconnects and reconnects

Sends a reboot command. Configuration changes that require a reboot will take effect after restart. The device will disconnect briefly then rejoin the mesh. Reboot after saving config changes.

seconds (0-60)
SHUTDOWNPower off — requires physical restart

Sends a shutdown command. The device will power off completely. It will only restart if physically power-cycled. Only use on devices with a reliable power source reset (e.g. solar nodes).

seconds
FACTORY RESET⚠ IRREVERSIBLE — All config erased

Permanently erases ALL settings on this device: all channel configurations, owner identity, LoRa settings, position config, MQTT config, security keys. The device must be configured from scratch using physical access. This cannot be undone.

PURGE NODE DATABASE⚠ IRREVERSIBLE — All node history erased

Removes all stored node information from this device's internal database. The device will need to rediscover all nodes through normal mesh operation. Channel settings and device configuration are preserved — only node records are cleared.
